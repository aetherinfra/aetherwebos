# Code of Conduct

## Our Commitment

The Aether project is committed to providing a welcoming, respectful, and collaborative environment for everyone participating in the community.

We value constructive discussion, mutual respect, and a shared interest in systems research and low-level engineering.

All contributors, maintainers, and participants are expected to uphold this Code of Conduct when interacting within the project.

---

## Expected Behavior

Participants in the Aether community are expected to:

- Be respectful and considerate in communication
- Provide constructive feedback during discussions and reviews
- Assume good intentions from other contributors
- Be open to technical discussion and differing perspectives
- Focus on improving the project and helping others learn

Collaboration and respectful communication help maintain a productive development environment.

---

## Unacceptable Behavior

The following behaviors are not acceptable within the project community:

- Harassment, discrimination, or personal attacks
- Insults or derogatory comments directed at individuals or groups
- Trolling, deliberate disruption, or inflammatory behavior
- Publishing private information about others without permission
- Any conduct that creates a hostile or unsafe environment

These behaviors may result in moderation actions.

---

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may take appropriate action in response to behavior that violates these guidelines, including:

- issuing warnings
- removing inappropriate comments or contributions
- restricting participation in project discussions or repositories

Decisions will be made with the goal of protecting the health of the community.

---

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub repositories
- issues and pull requests
- discussions
- project communication channels

It also applies when an individual represents the Aether project in public spaces.

---

## Reporting Issues

If you experience or witness unacceptable behavior, please report it to the project maintainers through GitHub issues or by contacting a repository maintainer.

All reports will be reviewed and handled appropriately.
